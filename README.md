# xs-docs

该文档是 [x-spreadsheet](https://github.com/myliang/x-spreadsheet) 库的文档开发库

[文档地址](https://hondrytravis.github.io/x-spreadsheet-doc/)

## 欢迎协作补充文档

首先你需要安装 [yarn](https://yarn.bootcss.com/) 的环境

```bash
  # 初始化
  yarn install
  # 项目启动
  yarn docs:dev
  # 然后开始愉快的开始协作吧~
```
