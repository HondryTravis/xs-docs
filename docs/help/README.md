# 如何协作编写文档

## 你需要安装 `yarn` 环境

首先你需要安装 [yarn](https://yarn.bootcss.com/) 的环境

```bash
  git clone <repo>
  # 初始化
  yarn install
  # 项目启动
  yarn docs:dev
  # 然后开始愉快的开始协作吧~
```
